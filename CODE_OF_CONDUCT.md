# Code of Conduct

Koml follows the **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its community code of conduct.

The Covenant is a widely-adopted standard text; we adopt it by reference rather than reproducing it here so that updates to the canonical version flow through automatically.

## Scope

This Code applies to all project spaces — GitHub issues, pull requests, code reviews, discussions, releases, and any other forum that represents Koml in public.

## Reporting

Concerns about conduct in any of these spaces can be sent privately to **debdut.saha.1@gmail.com**. Reports are read by the maintainer and handled in confidence.

## Enforcement

Maintainers will follow the Covenant's [enforcement guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) when responding to reports — the goal is to resolve issues constructively while keeping participation safe and useful for everyone.
